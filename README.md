# LiquidPlot

# 📚简介
本项目为Qt实现水波进度条。

# 📦软件架构
- Qt 5.9 + msvc 2015
- Windows(x32, x64)/Linux(x32, x64) 
- 理论上Qt 5.6以上都支持

# 🛠️主要技术
| 模块                |     介绍                                                                          |
| -------------------|---------------------------------------------------------------------------------- |
| signal\slot        |     控件、窗体间通信，事件处理                                               |
| QPainter        |     窗口的绘制                                          |
| QFont        |     时间字体                                          |
